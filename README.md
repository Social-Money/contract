# social money
## Before Getting Started
我们先安装必要的依赖
```bash
yarn # yarn is recommended as we have yarn lockfile
# or 或
npm i # 😔 npm is too old for package management
```

然后...... 开始测试构建
```bash
yarn compile
# or 或
npm run compile
```

## Test
```bash
npm run test-local
```
